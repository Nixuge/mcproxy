MCProxy

What's that for?

    MCProxy is a "proxy" that allows you to download Minecraft versions from networks that have the Mojang servers blocked.
    It's made primarily for the PolyMC and PrismLauncher launchers.
    You can get the source for the server here, it's fairly simple so feel free to dig into the code and selfhost your own server.
    Note that this doesn't restore the authentification servers, it only allows you to download the versions jars/libs jars.

Installation instructions

  • Download either PolyMC or PrismLauncher
    • IF POSSIBLE, GET THE PORTABLE BUILD (.zip).

  • Download the accounts.json file and place it in the PolyMC/PrismLauncher data folder
    • if portable, place it in the extracted folder with the executable
    • if not portable, the data folder should be in your %localappdata% on windows and on your .local/share on linux

  • Open the settings, go to the APIs tab, and inside "Metadata Server", place one of the URLs below:
    • If using PolyMC, https://mcdl.nixuge.me/get_data/meta.polymc.org/v1/
    • If using PrismLauncher, https://mcdl.nixuge.me/get_data/meta.prismlauncher.org/v1/

  • After these steps, the installation itself should be done
    Just add a new account with your username (top right>manage accounts>add offline) and you're good to go
